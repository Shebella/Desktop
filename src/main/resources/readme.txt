
  Safebox

======================================================================


Download
========
http://192.168.213.10/safebox


Installation
============
For windows users:
Launch Safebox-setup-x.xx.exe and setup safebox step by step.

For Linux users:
Make sure Java JRE have been installed and you are a superuser.
chmod a+x ./SafeBox-setup-linux-x.xx.bin
./SafeBox-setup-linux-x.xx.bin

Parameters
==========
	-help
	-web
	-exit
	-folder
	-test_connection

Notes
=====
1. Win7 users please run Safebox by Administrator role.
2. Ubuntu users please sudo the installing instruction beforehand.
